## 标准赞助位

| SKU | 赞助位 | 包含内容 | 标价（USD/月） | 适用场景 |
|-----|-----------|----------|--------------------:|----------|
| **SP-TABLE** | 对比表置顶行 | 单一品类对比表首行；「Sponsored」徽章；行高亮；该 SKU Modal 内披露 | **$400** | 旗舰帐篷/天幕/睡袋推广 |

## 资格与排除

**符合资格**

- 品牌在所选品类中已有产品收录于我们的官方目录。
- 赞助 SKU 尽可能使用 **verified** 或 **published** 数据。
- 品牌书面批准活动素材（见合同 §4）。

**不符合 / 我们保留拒绝权**

- 暗示虚假「第一」排名或篡改规格的展示。
- 尚未公开发布的品类（除非预先约定上线日）。
- 同一置顶行多家竞价 *（先签约者优先；可排队等候）*。

## 活动配置（技术说明）

| 字段 | 用途 |
|-------|---------|
| `productId` | 置顶 SKU 在官方目录中的 ID |
| `tier` | `table-featured` 或 `modal-featured` |
| `label` | 默认 `Sponsored`（可定制，如 `Partner`） |
| `expiresAt` | 自动下架日期（YYYY-MM-DD） |
| `campaignId` | 品牌内部 PO / 活动编号 |

配置位于 `data/sponsors.json`；变更在一次部署周期内生效。

## 付款与取消

| 条款 | 政策 |
|------|--------|
| **账单** | 按月预付，或季度预付（享 10% 折扣） |
| **付款方式** | 银行转账（USD）、PayPal、Wise *（在合同中指定）* |
| **取消** | 任一方须于下一计费周期前 **14 天书面通知** |
| **发布方提前终止** | 若无法交付展位，未使用整月按比例退款 |
| **赞助商提前终止** | 当前周期不退款；定制创意可收取 kill fee |
